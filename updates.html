2025 - Reading news like grandpa in 2025

I started to experiment with a small project called Morning Nerds using a ReMarkable device for reading news. The goal was to escape the noise of social media and create a peaceful way to consume tech news and updates.



Social Media and the noise

I used to check news (tech, world, local news) on Mastodon and Bluesky. I mainly use Mastodon but I follow some other users on bluesky. It's convenient to have all in Mastodon but there is also a lot of mixed content and too much noise to get the data I want.

So I started to prototype with a new tech call RSS, yes RSS is pretty old but still doing the job and is better than other social media. I removed all news I could access with RSS feeds and keep only people I am interested in. The result is a pdf I generate daily(ish) from rss feeds with summaries. It's really nice, and peaceful to read, especially when you go outside in a park or a coffee shop. Also it changed how I interact with social media, I don't go as often as before.

How it works

The setup is straightforward, I have a config file with RSS feeds. Everyday I generate a PDF file with the following steps:

  • fetch new articles
  • Create a summary with a llm
  • Add a few image to illustrate if I find an image in the article
  • And build the PDF with a template

Why PDF format?

On reader device epub format works but it looks like the subset of html/css is not that easy for all reader devices and some devices clearly cut some features. So to generate a nice layout like having 2 columns I need to use PDF.

What's next

I would like to add content from social media for people I am following but filtered. Right now I am following mainly Graphics Programming developers, but even in this community people express themself (and dont stop we are humans) but for technical news I want to extract only tech informations from cats, politics, boats (you pictures are great btw :) )

My current system works only to get an idea of what's going on, it does not give full details. For full details I still need to be on the computer click on links in the PDF. Ideally I would like to mark things on the ReMarkable so I could have this list on my computerâ€¦ but right now, the Remarkable API is too limited. I would love to automatize this part but maybe when there will be a richer API on the device.

Conclusion

I don't know if I am the only one annoyed by the waste of time growing to get valuable content on the web, but when I was prototyping this tool I felt insane the amount of garbage we need to eat to get the real content. I understand now when my grandpa was reading the newspaper outside with his coffe, it's a really peaceful routine.
